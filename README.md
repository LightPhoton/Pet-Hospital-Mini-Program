# Pet-Hospital-Mini-Program
宠物医院微信小程序

前端技术：微信小程序原生组件，Vant-app开源组件，diygw可视化工具

后端技术：java1.8，springboot2.6，mybatis，MySQL5.7，tomcat

其他：阿里云oss对象存储

前端导入微信开发者工具，修改appid，
后端代码导入IDEA，修改Wx_userController的appid和appScrect，修改FileServiceimpl的阿里云oss配置信息即可使用
